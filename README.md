Telegram Fitness Bot
A telegram bot for training assistance with exercise programs and music playlists for the gym.

Description:
This bot is designed to help you organize your workouts. It provides access to ready-made workout programs and playlists for gym classes through the user-friendly Telegram interface.

Capabilities:
Workout Programs - Get access to various exercise programs
Gym Playlists - a selection of music playlists for motivation during workouts
Easy navigation - simple menu with buttons for quick access to all functions
